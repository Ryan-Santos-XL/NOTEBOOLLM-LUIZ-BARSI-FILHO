O Livro-Razão da Riqueza

> Projeto baseado na masterclass de Luiz Barsi sobre investimentos, construção de patrimônio e geração de renda através de dividendos.

Sobre o projeto

Este repositório reúne os materiais utilizados na masterclass **"O Livro-Razão da Riqueza"**, apresentando os principais conceitos do método de investimento desenvolvido por Luiz Barsi.

O objetivo é organizar os conteúdos estudados, facilitando futuras consultas e servindo como material de apoio para quem deseja compreender os fundamentos do investimento em empresas pagadoras de dividendos.
